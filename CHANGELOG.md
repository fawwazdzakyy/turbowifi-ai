# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-17
### Added
- **Core Orchestrator**: Completely autonomous network optimization engine.
- **TUI Dashboard**: Real-time terminal UI built with Textual (`turbowifi watch`).
- **MTU Optimizer**: Intelligent Path MTU discovery and safe fallback application.
- **Download Engine**: Accelerated segmented async downloader with Range requests.
- **DNS Optimizer**: Benchmark and automatic application of fastest DNS providers.
- **TCP BBR Support**: Enable congestion control optimization.
- **Auto Daemon**: Background monitoring with 24-hour intelligent cooldowns.
- **Doctor CLI**: Comprehensive environment diagnostics (`turbowifi doctor`).
