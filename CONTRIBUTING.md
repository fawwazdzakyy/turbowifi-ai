# Contributing to TurboWiFi AI

Thank you for your interest in contributing to TurboWiFi AI! 

## Getting Started

1. Fork the repository and clone it locally.
2. Install dependencies: `pip install -e .[dev,all]`
3. Run tests before submitting a PR: `pytest tests/`
4. Lint code: `ruff check src/` and `black src/`

## Pull Request Process

1. Ensure any new functionality includes tests.
2. Update the README.md with details of changes to the interface.
3. PRs must pass all GitHub Actions CI checks before merging.
